# Notification Sounds

Place notification sound files in this directory:

- `notification.mp3` - Default notification sound

Recommended specifications:

- Format: MP3 or OGG
- Duration: < 2 seconds
- File size: < 50KB
- Sample rate: 44.1kHz
- Bit rate: 128kbps

You can use free sounds from:

- https://freesound.org/
- https://notificationsounds.com/
- https://zapsplat.com/
